# 🥔 Potato Disease Classification (Deep Learning)

## Overview
This is an end-to-end deep learning web application that classifies
potato leaf diseases from images using a Convolutional Neural Network (CNN).

The goal is to help identify plant diseases quickly using computer vision.

## Technologies Used
- Python
- TensorFlow / Keras
- Convolutional Neural Networks (CNN)
- FastAPI
- React

## How It Works
1. User uploads an image of a potato leaf
2. The image is sent to a FastAPI backend
3. A trained CNN model predicts the disease
4. The prediction result is returned to the frontend

- Healthy
- Early Blight
- Late Blight
